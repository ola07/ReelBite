# Changelog

All notable changes to ReelBite are documented here.

## [1.0.0] - 2026-03-18

### Core Platform
- Full-screen vertical video feed with TikTok-style snap scrolling
- Video playback with mute/unmute toggle
- Double-tap to like with animated heart overlay
- Video action sidebar: Like, Comment, Bookmark, Share
- Video overlay: Creator info, restaurant name, dish name, description
- Feed category filters: For You, Trending, Near Me, Following

### Restaurant Discovery
- Dual-mode explore screen: Cuisine and Discover modes
- Cuisine filtering: 20 cuisine types
- Discovery filters: 10 vibes, 6 moods, 8 cravings, 8 dietary options
- Local highlights: Hot Right Now, Rising Stars, Most Reviewed, New Arrivals
- Restaurant search bar
- Restaurant grid with ratings, cuisine, price level

### Restaurant Details
- Three-tab layout: Menu, Reviews, About
- Video-enhanced menu items with video previews
- Categorized menu with prices, descriptions, tags, calories
- Customizable menu items with price deltas
- Review listings with star ratings and user profiles
- Restaurant about section with hours, location, contact

### Ordering System
- Cart management: Add, remove, adjust quantities
- Per-item customizations and special instructions
- Delivery and pickup order types
- Pricing: Subtotal, tax (8.875%), delivery fee ($3.99), tip
- Order checkout flow
- Order history with active/past tabs
- Order status tracking (placed through delivered)

### Reservation System
- Multi-step booking: Date, Party Size, Time, Confirmation
- Party sizes 1-12 guests
- Time slot selection
- Special requests
- Confirmation code generation
- Reservation status tracking

### Creator Tools
- Creator profile with video grid gallery
- Verified badge system
- Tiered creator levels: Rising, Established, Elite, Verified Critic
- Follower/following system
- Creator tools navigation hub

### Creator Analytics
- Period selector: 7 Days, 30 Days, 90 Days
- Overview: Views, likes, new followers, avg watch time
- Peak engagement times with hourly bar chart
- Engagement by day chart
- Audience demographics: Age, gender, top cities
- Content performance rankings

### Creator Earnings
- Total earnings with period comparison
- Pay-per-sale tracking
- Top performing videos by earnings
- Recent sales transaction feed
- Affiliate commission tracking per restaurant
- Payout system with minimum threshold

### Creator Collaborations
- Collaboration types: Sponsored Review, Featured Video, Menu Launch, Event Coverage
- Summary: Total earned, active, pending
- Accept/Decline workflow for pending offers
- Messaging for active collaborations
- Status tracking: Pending, Active, Completed, Declined

### Community Features
- Community badges: Trending, Local Favorite, Community Pick
- Written review cards (mini-blog) attached to videos
- Star ratings on review cards
- Expandable review content

### Authentication
- Welcome/onboarding screen
- Email/password login and signup
- Session persistence
- Auto token refresh

### User Profile
- Profile with avatar and stats
- Quick stats: Favorites, Reservations, Orders, Reviews
- Menu: Favorites, Saved Videos, Reservations, Orders, Reviews
- Settings and sign out

### Design System
- Dark emerald green theme
- Glassmorphism effects with blur
- Custom blur tab bar
- Consistent spacing and typography scale
- Lucide icons throughout
